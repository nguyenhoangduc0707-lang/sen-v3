========================================
🚀 HƯỚNG DẪN BẮT ĐẦU NHANH (DYT Core - Facebook Auto Poster)
========================================

1. FIX MÔI TRƯỜNG (Python + Terminal):
   - Cài Python 3.11+ từ python.org (Windows installer).
   - **BẮT BUỘC**: Trong installer, tick "Add Python to PATH".
   - Nếu đã cài rồi: chạy lại installer → Modify → tick Add to PATH → Repair.
   - Kiểm tra: python --version  (phải in ra version, không "not found").
   - Windows Terminal: Tải bản x64 (không phải arm64) từ GitHub Microsoft/terminal.
     - Dùng PowerShell là đủ, không cần Terminal Preview nếu bị lỗi "can't run on your PC".

2. LƯU AUTH FACEBOOK (quan trọng nhất cho bot):
   - Chạy: python save_facebook_auth.py
   - Browser mở → Login FB thủ công (bằng cách bạn hay dùng: số ĐT, approve app...).
   - Sau login: Click vào Trang của bạn → Click "Đăng với tư cách Trang" nếu hiện.
   - Đợi thấy "Viết điều gì đó lên trang..." (ô soạn cho Trang).
   - CHỈ KHI ẤY mới nhấn Enter trong console để save facebook_auth.json .
   - Chạy 2-3 lần để profile "quen" Meta (giữ .fb_profile_poster).
   - Test: python test_facebook_auth.py  (nếu cần, click switch thủ công trong browser mở).

3. LẤY LINK AFFILIATE & TẠO BÀI:
   - Truy cập: https://pub.accesstrade.vn
   - Đăng nhập → Campaigns → Get Link.
   - Chạy schedule: python schedule_optimized_posts.py --fanpage_key affiliate_fashion_cosmetics --days 1
   - (Tương tự cho motivational_postcard)

4. CHẠY BOT ĐĂNG BÀI:
   - python run_scheduled_posts.py --once   (đăng các bài đã schedule đến giờ)
   - Hoặc full core: powershell -ExecutionPolicy Bypass -File .\start_core.ps1
     (3 terminal: API, Worker, Scheduled Poster)

5. KIỂM TRA:
   - python check_posts.py
   - python analyze_performance.py

========================================
💡 LƯU Ý QUAN TRỌNG CHO FACEBOOK BOT:
- Phải login thủ công và switch sang "Đăng với tư cách Trang" trong save script.
- URL phải đúng từ config/fanpages.json (profile.php?id=... của bạn).
- Nếu "This content isn't available": Chưa login đúng / không phải admin / page không tồn tại.
- Test bằng tay: Mở Chrome (đã login) → vào đúng URL → xem có "Viết điều gì đó lên trang..." không.
- Nếu vẫn fail sau khi test pass: Chạy save lại vài lần.
